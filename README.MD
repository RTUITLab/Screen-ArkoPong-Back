# arkopongBack #
This project is SignalR backend for [ScreenArkoPong][Game]

---
##### Instruction manual #####
* Download the latest version;
* Open `\arkopongBack` folder in the console;
* Use `dotnet run` command;
* Wait.
---
#### Requirements ####
* .NET 5.0

[Game]: https://github.com/RTUITLab/Screen-ArkoPong
